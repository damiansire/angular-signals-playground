Antes de usar signals conviene entender cómo Angular detecta cambios y los actualiza en pantalla. Estos ejemplos van del DOM al modelo de reactividad.

  • Sub-nivel 1: el HTML como árbol de nodos (DOM).
  • Sub-nivel 2: cómo una variable se refleja en la vista (HTML → JS → Angular).
  • Sub-nivel 3: la detección de cambios "clásica" (Zone.js avisa que pasó algo async; entonces Angular recorre el árbol).
  • Sub-nivel 4: la detección con signals, mucho más quirúrgica.
